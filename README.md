# Defense Contractor Air Emissions Analysis

## Project Overview
This project analyzes air emissions data from major U.S. defense contractors using EPA TRI 2024 data.

## Tools Used
- Python (Pandas, Matplotlib)
- MySQL
- Jupyter Notebook

## Key Analysis
- Total emissions by company
- Top 10 polluting chemicals
- Emissions by state
- Top polluting facilities

## Database Design
Data was cleaned in Python and imported into MySQL for SQL-based analysis using aggregation and window functions.
